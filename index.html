Tag: H1 até H6

Dando inicio ao meu resumo sobre tags que aprendi até o momento,

aprendi que o H1 usamos como titulo.

Tag: P

Tag "P" utilizamos como parágrafo,podemos então escrever utilizando junto com H1 até H6

como uma tag de escrita.

Tag: mark

Tag mark utilizamos como uma formar de destacar determinada palavra.

Tag: small

A tag "small" utilizamos para deixar qualquer palavra ou texto pequeno.

Tag: i

A tag "i" deixamos qualquer palavra ou texto italico.

Tag: u

Assim como temos algumas tags de destacar a tag U tem como finalidade sublinhar

Tag: strong

Mais uma tag de destaque e ela tulizamos para deixar algo em negrito.

Tags: ul e ol

Aqui temos duas tags "ol" ordenada e a "ul" não ordenada, a OL utilizamos

como uma lista ordenada já a UL é uma tag de lsita não ordenada não tem representação numerica.

Tags: LI, A e HR

A tag HR utilizamos como quebra a de linha, é usada para criar uma âncora de onde vincular, o atributo href é usado para onde

endereçar o documento, e as palavras entre as tags de abertura e fechamento da âncora serão exibidas como um hipervínculo.

Tags: blockquote, sup e sub

Blockquote indica que o texto incluído é uma longa citação, sup é utilizado escrever um texto com letras menores e

com o alinhamento acima das que do resto do conteúdo,Sub é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo.
